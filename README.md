# CS2 Mod - Модификация для Counter-Strike 2

## Описание проекта

CS2 Mod - это клиентская модификация для Counter-Strike 2, предоставляющая дополнительные возможности через overlay-интерфейс.

## Структура проекта

```
Cs2-mod/
├── loader/                 # EXE проект - загрузчик модификации
│   ├── src/               # Исходный код загрузчика
│   ├── include/           # Заголовочные файлы
│   └── resources/         # Ресурсы загрузчика
├── mod/                    # DLL проект - сама модификация
│   ├── src/               # Исходный код модификации
│   │   ├── core/          # Ядро модификации
│   │   ├── overlay/       # Overlay система
│   │   ├── renderer/      # Рендеринг интерфейса
│   │   ├── steam/         # Интеграция со Steam
│   │   └── utils/         # Утилиты
│   ├── include/           # Заголовочные файлы
│   └── resources/         # Ресурсы модификации
├── config/                 # Конфигурационные файлы
├── docs/                   # Документация
└── resources/              # Общие ресурсы
```

## Компоненты

### Loader (Загрузчик)
- Запуск CS2 через Steam API
- Инжектирование DLL модификации в процесс игры
- Проверка актуальности модификации

### Mod (Модификация)
- Overlay-интерфейс поверх игры
- Система рендеринга (ImGui/DirectX)
- Обработка ввода (клики, нажатия клавиш)
- Конфигурация настроек

## Требования

- Windows 10/11
- Visual Studio 2022
- CMake 3.20+
- Counter-Strike 2
- Steam Client

## Сборка

```bash
mkdir build && cd build
cmake ..
cmake --build . --config Release
```

## Использование

1. Запустите `loader.exe`
2. Дождитесь запуска CS2
3. Модификация автоматически загрузится
4. Нажмите клавишу INSERT для открытия/закрытия меню

## Безопасность

⚠️ **Внимание**: Данная модификация предназначена исключительно для образовательных целей. Использование модификаций в сетевых играх может нарушать правила игры и привести к блокировке аккаунта.
